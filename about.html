About The Map

Solana History Map is a public visual atlas for ecosystem memory: outages, rebuilds, security scars, liquidity infrastructure, airdrops, mobile experiments, and meme culture.

Scope

The grant-facing artifact is the map, landmark data, source index, and shareable history cards. It is not a request to fund token liquidity, promotion, market-making, or trading.

Origin

The artwork began inside MEOW vs WOOF, a paused cats-vs-dogs token battle experiment. The original experiment remains available as an archive so the history is visible instead of hidden.

Roadmap

  • Complete all 22 sourced landmark entries.
  • Publish open landmark data for reuse.
  • Add mobile polish, accessibility, and performance passes.
  • Create shareable landmark cards for community education.

No Affiliation Claim

The map references projects, incidents, and ecosystem events as historical landmarks. It does not claim endorsement, partnership, sponsorship, or official status with any referenced project.
