# Changelog

Notable changes to the Solana History Map site and dataset.

## 2026-07

- Dataset v1.0.0: per-landmark `lastVerified` dates and `tier` labels
  (playable teardown vs illustrated timeline); versioned `landmarks.json`.
- Open-source scaffolding: LICENSE (MIT), LICENSE-DATA (CC BY 4.0), README,
  CONTRIBUTING with the factual-correction flow, issue template.
- Removed the retired memecoin-era archive (token battle page, token T&C,
  whitepaper, legacy scripts) — the origin stays documented in the footer,
  the About page, and git history.
- Clarity pass on all interactive simulations (visible narration line,
  fixed dead-ends, Jupiter routing now genuinely beats a hand-route).
- New origin mini-game THE TRUCE (replaces the emoji tug-of-war): the two
  mascots became co-keepers of the archive.
- Full source re-verification of all 22 landmark records (73 references).

## 2026-06

- All 22 landmarks upgraded with bespoke or deep-dive interactives across
  three batches; panel-clearance and marker-pin map fixes.
- Mobile feed mode (browse/filter/detail-sheet) for small screens.
- HD map re-export from the clean master; per-landmark hero art cropped
  from the full-resolution map.
- Per-landmark static pages, OG share cards, open dataset, source index,
  and sitemap generated from a single source of truth (landmarks-data.js).

## Earlier

- Project pivoted from the paused cats-vs-dogs meme experiment (meow-woof)
  into a source-cited Solana history atlas. The transition is preserved in
  git history.
